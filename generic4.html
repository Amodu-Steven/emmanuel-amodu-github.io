PROGRAMMING ABB ROBOTS

This project involved programming an ABB IRB 1200 industrial robot using ABB RobotStudio software to perform precise drawing tasks. The robot was programmed to draw geometric shapes (dodecagon, nonagon, square), the word "SALFORD," and the programmer’s name "AMODU" using an attached pen.

Project Title:

Programming and Simulation of ABB IRB 1200 Robot

Tools Used:

ABB RobotStudio, Flex Pendant Simulator, SolidWorks (for CAD models)

Overview:

This project involved programming and simulating an ABB IRB 1200 industrial robot to autonomously draw shapes (dodecagon, nonagon, square), write the word "SALFORD", and trace the name "AMODU" using a pen tool in a virtual environment.

Key Activities and Skills Demonstrated:

Robot Programming:

Developed routines using ABB’s RAPID programming language, employing motion instructions (MoveJ, MoveL, MoveC) to control robot movement for linear, circular, and non-linear paths.

Coordinate Systems and Calibration:

Configured base, world, tool center point (TCP), and work object coordinate systems to ensure precise path execution on various work surfaces.

Zone Data and Velocity Management:

Applied different ZoneData values (Fine, Z0, Z1) and adjusted velocity settings to optimize robot accuracy and speed during tasks.

CAD Integration:

Imported custom CAD models (e.g., a pen tool and nameplate) into RobotStudio for accurate simulation and tool alignment.

Interface Development:

Created an interactive user interface on the Flex Pendant to allow users to select between drawing shapes, writing "SALFORD," or writing "AMODU."

Simulation and Execution:

Successfully simulated and manually executed the programs, verifying robot behavior in a virtual environment that mimics real-world conditions.

Impact and Outcomes:

  • Developed strong practical skills in industrial robot programming and offline simulation.
  • Gained hands-on experience using ABB’s RobotStudio, including tool calibration, coordinate setting, and program testing.
  • Built a deep understanding of path planning, motion control strategies, and robotic system configuration.
  • Acquired techniques to prepare programs for both simulation testing and real-world deployment.

Conclusion:

Through this coursework, I enhanced my ability to program industrial robots efficiently, utilize simulation tools to optimize performance before deployment, and apply robotics principles critical to automation industries such as manufacturing, logistics, and assembly.
