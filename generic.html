AUTONOMOUS MOBILE ROBOT NAVIGATION

I designed and implemented an autonomous navigation system for the Pioneer 3-DX mobile robot, demonstrating my expertise in robotics, algorithm development, and data analysis. This project involved designing and testing a navigation architecture that enabled the robot to move autonomously in a controlled environment while reaching predefined goals.

To accomplish this, I developed and tested three key navigation behaviors:

  • Odometry-Based Navigation: The robot followed a series of waypoints using an odometry model that incorporated angular and linear transition algorithms.
  • Target Tracking: The robot identified and followed a moving object or individual in real time.
  • Obstacle Avoidance: The robot dynamically detected and avoided obstacles to ensure safe navigation.

The experiment was conducted in a living lab environment, where the robot navigated through predefined goal points under both controlled and dynamic conditions. Data analysis included Euclidean distance calculations, correlation analysis, and error thresholding to optimize movement accuracy. The navigation system was designed using dead reckoning principles, where wheel encoder sensors provided trajectory and movement data. I implemented mathematical models to calculate angular transitions (heading adjustments) and linear transitions (forward movement), ensuring accurate goal-directed navigation.
