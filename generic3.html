3D INTERACTIVE VISUALIZATION

Project Title:

Aztec Island – An Educational 3D Interactive Virtual Environment

Overview:

Developed an immersive 3D educational game using Unity and C# programming. The virtual world allows players to explore an Aztec-themed island, learn about history, and interact with dynamic game elements.

Key Features:

  • Three Main Scenes: Menu, Prologue, and Game Scene featuring a maze and historical elements.
  • Movement and Navigation: First-person controller with collision detection and realistic interaction using Unity’s physics system.
  • Interactive Elements: Triggered doors, collectible Aztec rings, animated obstacles, and responsive environment using C# scripts.
  • Audio and Graphics: Integrated environmental sounds, background music, dynamic lighting, and realistic textures.

Technical Details:

  • Unity Version: 2022.3.8f1
  • Scripting Language: C#
  • 3D Models and Textures: Imported from Unity Asset Store and custom-built
  • Collision and Physics: Box, Sphere, and Mesh colliders with trigger zones

Experimental Design:

  • Designed tasks including maze navigation, ring collection, and emergency beacon activation.

Impact and Outcomes:

  • Strengthened skills in Unity game development, UX design, and experimental research.

The project successfully blends interactive entertainment and educational content, providing a dynamic learning experience that captivates users and enhances their understanding of historical and cultural subjects.
